# Assets do app

- **icon** – Ícone simples do aplicativo (ícone do app na home do dispositivo).
- **logo** – Logo completa usada na abertura do app (splash / tela inicial).

Referência da identidade visual: arquivos em **sintonia-backoffice/public/assets/** (icon e logo).  
Copie ou adapte para aqui (ex.: `icon.png`, `logo.png`) para manter a mesma identidade no mobile.
